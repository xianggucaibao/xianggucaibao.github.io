Regression - Predicting Possum Head Length¶

The possum dataset consists of nine morphometric measurements and additional details such as sex and age on each of 104 mountain brushtail possums, trapped at seven sites from Southern Victoria to Central Queensland. The dataset was originally part of the DAAG R package and is available on Kaggle.

If we leave the site column as is, our ML model will assume that there is an inherent order to the sites, when really the numbers serve to distinguish one site from another and can be thought of as different site names. This nominal data (data that cannot be ordered) is best represented using one-hot-encoding:

min_mse = float("inf")
error_increased = 0

for n_estimators in range(1, 100):
    gbr.n_estimators = n_estimators
    gbr.fit(x_train, y_train)
    
    y_pred = gbr.predict(x_test)
    mse = mean_squared_error(y_test, y_pred)
    
    print('Number of Estimators: ', gbr.n_estimators_)
    print('MSE: ', mse)
    
    # if an added estimator improves the model, the MSE falls
    # if we add 10 estimators in a row that don't improve the MSE, we stop adding estimators
    # the number of estimators minus 10 then gives us the best model: n_estimators - 10
    
    if mse < min_mse:                
        min_mse = mse
        error_increased = 0
    else:
        error_increased += 1
        if error_increased == 10:
            break
